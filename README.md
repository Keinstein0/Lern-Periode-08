# Lern-Periode-08

Um ein Relevantes Thema zu finden überlegte ich zuerst was genau ich neues lernen will. Die 2 interessantesten Themen fand ich 1. Neue Programmiersprache und 2. ein Game Engine. Ich entschied mich dagegen eine neue Programmiersprache zu lernen da ich lieber etwas lokales grafisches machen wollte. Daher entschied ich mich für ein Gameengine. Da ich schon einmal Unity ausprobiert habe und es eher Verwirrend fand (mit dem GUI und level editor) wollte ich nach einem Framework suchen in welchem alles auf Code aufgebaut ist. Eine Python library mit der ich schon mal gearbeitet habe kam mir in den sinn (pygame) jedoch war diese sehr limitiert und langsam. Daher suchte ich nach einer alternative und fand dank diesem reddit post (https://www.reddit.com/r/pygame/comments/farqs5/alternatives_to_pygame/) MonoGame. Ich schaute mir die Fähigkeiten dieses Frameworks an (Codebeispiele) und fand dass dieses Framework sehr interessant klang. 

Als Tutorial werde ich (https://www.youtube.com/watch?v=it_vSf80sSk&list=PLvN4CrYN-8i4MhiXQMajViJoC5udFfNfA) verwenden. Wärend es mich nicht zu einem fertigen Game bringt werde ich das Turorial durcharbeiten und alle elemente in ein Game (simpler plattformer) integrieren.
